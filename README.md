##Development:

1. create `.dev` file

    ```
    coreMysqlUser=
    coreMysqlPassword=
    coreMysqlHost=
    coreMysqlDatabase=

    ``` 
1. `npm run start.local`

1. api explorer `http://localhost:3000/explorer`

1. copy and paste the `id` field in the response to "Set access token" at the top right 

###Postman

1. import `http://localhost:3000/explorer/swagger.json`
